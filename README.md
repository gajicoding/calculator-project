# CH 2: 계산기 과제

## 단계별 목표
- **클래스를 사용하지 않는 계산기:**
  - [level 1] : `필수 구현` 클래스 없이 자바의 기본 문법만을 사용하여 구현한 계산기


- **클래스를 활용한 계산기:**
  - [level 2] : `필수 구현` 객체 지향 개념을 적용하여 클래스 정의, 연산을 수행하는 메소드 작성한 계산기
  - [level 3] : `도전 구현` 자바의 Enum, 제네릭, 람다 & 스트림을 적용한 계산기


## 파일 구조
```bash
src
├──── level1
│  └──── App.class
├──── level2
│  ├──── App.class
│  ├──── Const.class
│  ├──── Calculator.class
│  └──── MyScanner.class
├──── level3
└──── README.md
```
